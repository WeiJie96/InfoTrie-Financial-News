Please find the crawler in news\news\spiders\news.py

Install Scrapy version 1.4.0

To run the crawler enter this into the command prompt:

scrapy crawl News -o name.csv -s USER_AGENT="Mozilla/5.0 (Windows NT 6.1) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/41.0.2228.0 Safari/537.36" -s FEED_FORMAT="csv"
